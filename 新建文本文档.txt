1、在github上创建储存库
2、将云端的仓库克隆到本地 
   git clone https://github.com/YY-xiaolei/computer.git
3、先将页面复制到克隆好的文件夹
4、打开克隆好的文件夹
   cd 文件夹名称
5、添加内容
   git add *
6、查看状态
   git status
7、标记
   git commit -m "第一次上传"
8、上传
   git push
